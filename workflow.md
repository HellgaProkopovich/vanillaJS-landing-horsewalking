# How create commits

what(where)/description

## `feat` — new feature

feat: add header layout
feat: implement mobile menu
feat(section-contacts): add contact form validation

## `fix` — fixing

fix(header): fix menu overlap on mobile
fix: correct form submit error
fix(section-about): fix slider navigation bug

## `style` — changings in styles

(without logic changes)

style: improve button hover styles
style: adjust container paddings
style: fix typography spacing

## `refactor` — переработка кода без изменения поведения

refactor: simplify menu logic
refactor: split styles into modules

## `chore` — служебные изменения

chore: add gitignore
chore: configure sass build
chore: update dependencies

## `docs`

docs: add README
docs: update project setup instructions
